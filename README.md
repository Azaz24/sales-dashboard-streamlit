# 📊 Sales Dashboard App

##  Overview
This is an interactive sales dashboard built using Streamlit. It allows users to filter sales data by category and visualize trends.

##  Features
- Filter sales data using a dropdown
- Display filtered data in table format
- Visualize sales using a line chart

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run sales_app.py
